# FCC (Federal Communications Commission)

The Federal Communications Commission provides free public REST APIs for radio station license lookup, broadband coverage maps, spectrum auction data, broadcast contours, census block conversions, consumer complaint databases, electronic comment filings, and regulatory data.

## APIs

- **License View API** — FCC-issued radio license data across spectrum bands
- **Spectrum Dashboard API** — Spectrum ownership overview (225 MHz–3700 MHz)
- **Area and Census Block API** — Census block data by lat/lon coordinates
- **Contours API** — FM/TV broadcast service area contours
- **ECFS API** — Electronic Comment Filing System regulatory filings
- **Consumer Complaints API** — Consumer complaint data (phone, internet, TV)
- **Public Inspection Files API** — Broadcast and cable public inspection files
- **Broadband Data Collection API** — ISP broadband coverage data
- **Content API** — FCC website news and document content
